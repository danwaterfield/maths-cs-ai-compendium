# Advanced Text Generation

- Text diffusion models: D3PM, MDLM, continuous text diffusion, score matching for discrete data
- Text OCR: scene text detection and recognition pipelines, CRNN, document OCR, vision-language OCR
- Controllable generation: classifier-free guidance for text, RLHF, DPO, constitutional AI
- Long-context methods: sparse attention, sliding window, ring attention, memory-augmented models
- Retrieval-augmented generation (RAG): retriever-reader architecture, dense passage retrieval, chunking strategies
- Speculative decoding, Medusa heads, parallel generation
